# 21st Dev Portfolio

This app is a Vite React frontend. The contact form submits directly to Web3Forms.

## Local development

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Import this repository into Vercel.
2. Set the project root directory to `21st-dev`.
3. Deploy.

## Security

- Web3Forms direct integration exposes the access key in frontend code.
- Use this only if you are comfortable with a public form key.
- Do not commit unrelated private `.env` files.
